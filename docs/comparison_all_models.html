Learn2Smile - Ablation Studies

Here we are comparing the results from our model vs. ablation models for 30 randomly chosen ground truth videos. The ablation models are sorted by their effectiveness using the "L2 Error w/ Shift" measure detailed in the paper.

On the left side:
The 1st column shows the ground truth of the user's facial expressions.
The 2nd column shows the ground truth response of the agent.
The 3rd column shows the predicted response from our model, which has two predictors for low-frequency and high-frequency components respectively, and uses both user's and agent's history as input signal.

On the right side:
The 4th column shows the predicted response from our model without frequency decomposition (i.e. without splitting the output into low-frequency and high-frequency).
The 5th column shows the predicted response from our model without using agent's history as input signal.
The 6th column shows the predicted response from our model without frequency decomposition, and without using agent's history as input signal.

Note that the 30 videos shown are randomly selected from the hold-out test set.

Please see comments below for findings in each group.

(For comparison with baselines, please see: link)

Group 1:
User, ground truth Agent, ground truth Agent, our model Agent, our model w/o frequency decomposition Agent, our model w/o agent history as input Agent, our model w/o frequency decomposition and agent history as input

Comment: In this group, we can notice that:
1. The predicted response from models without frequency decompositon (4th and 6th column) looks robotic and not natural. This is because these two models are not predicting the high-frequency component, and thus losing a lot of the subtle movements that come with it. For a detailed comparison between the agent's ground truth sequence and its corresponding low-frequency and high-frequency components, please see: link.
2. When the user (1st column) is not talking, we can see that both the agent's ground truth face (2nd column) and our model's predicted face (3rd column) start talking, which means that our model successfully learns the concept of taking turns during a conversation. Notice that the faces in 5th column and 6th columns have mouth movements as well, indicating that turn-taking is a relatively easy task to learn with these ablation models.

Group 2:
User, ground truth Agent, ground truth Agent, our model Agent, our model w/o frequency decomposition Agent, our model w/o agent history as input Agent, our model w/o frequency decomposition and agent history as input

Comment: In this group, we can notice that:
1. When the user (1st column) is talking, our model (3rd column) is able to predict head nods and smiles, which are very natural responses to a talking person. Our model is able to learn these concepts without explicit labels of these movements in the dataset.
